---

## Stack tecnológico

### Backend
- NestJS
- TypeScript
- Express (adapter)

### Base de datos
- SQLite
- Prisma ORM

### Documentación
- Swagger

### Calidad de código
- ESLint
- Prettier

### Testing
- Jest
- Supertest (E2E)
