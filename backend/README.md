# Backend – NestJS + Prisma (SQLite)

Backend desarrollado con **NestJS**, usando **Prisma ORM** con **SQLite**, documentado con **Swagger** y con un entorno completo de calidad de código (ESLint + Prettier) y testing (Jest).

---

## Stack tecnológico

### Backend
- NestJS
- TypeScript
- Express (adapter)

### Base de datos
- SQLite
- Prisma ORM

### Documentación
- Swagger

### Calidad de código
- ESLint
- Prettier

### Testing
- Jest
- Supertest (E2E)
